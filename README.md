# YetAnotherSSHClient

YetAnotherSSHClient — это простой и удобный SSH-клиент, написанный на Java с использованием Swing. Он предоставляет базовые возможности для управления SSH-соединениями, сохранения избранных серверов и настройки интерфейса.

## Особенности

- **Многовкладочный интерфейс**: Открывайте несколько SSH-сессий одновременно в разных вкладках.
- **Управление избранным**: Сохраняйте данные серверов (хост, порт, пользователь, пароль, файл ключа) для быстрого доступа через боковую панель или меню.
- **Поддержка аутентификации по ключу**: Возможность использования Identity File для подключения.
- **Настройка тем**: Поддержка Темной, Светлой и Gruvbox тем.
- **Настройка шрифтов**: Отдельные настройки шрифтов для интерфейса и терминала.
- **Безопасность**: Пароли в конфигурационном файле зашифрованы с использованием AES-128.
- **Нативный вид на Windows**: Генерация `.exe` файла и использование иконки приложения.

## Требования

- **Java 21** или выше.
- **Maven** для сборки проекта.

## Сборка и запуск

### Сборка JAR

```bash
mvn clean package
```
Собранный JAR-файл будет находиться в директории `target/`.

### Создание EXE (только для Windows)

Проект настроен на создание исполняемого файла для Windows с помощью `launch4j-maven-plugin`.
```bash
mvn clean package
```
После выполнения этой команды в директории `target/` появится `YetAnotherSSHClient.exe`.

### Запуск из исходников

```bash
mvn clean compile exec:java -Dexec.mainClass="main.Main"
```

## Использованные технологии

- **JediTerm**: Продвинутый виджет терминала.
- **Apache MINA SSHD**: Библиотека для работы по протоколу SSH.
- **FlatLaf**: Современный Look and Feel для Swing.
- **GSON**: Работа с JSON конфигурациями.
- **Lombok**: Минимизация шаблонного кода.
