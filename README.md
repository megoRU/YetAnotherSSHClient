# YetAnotherSSHClient

**YetAnotherSSHClient** — лёгкий SSH-клиент на Java (Swing) для быстрого подключения к серверам без лишней сложности. Подходит для повседневного администрирования и работы с несколькими хостами.

## Возможности

- **Многовкладочный режим** — несколько SSH-сессий одновременно
- **Избранные серверы** — хранение хоста, порта, пользователя, пароля или ключа
- **Аутентификация по ключу** — поддержка Identity File
- **Темы оформления** — светлая и тёмная
- **Гибкая настройка шрифтов** — отдельно для UI и терминала
- **Windows-сборка** — генерация `.exe` с иконкой приложения

## Требования

- **Java 21+**
- **Maven**

## Используемые технологии

- **JediTerm** — терминальный компонент
- **Apache MINA SSHD** — работа по протоколу SSH
- **FlatLaf** — современный Look & Feel для Swing
- **Gson** — сериализация конфигурации в JSON

## Конфигурация

Файл настроек хранится локально:

- **Windows:** `C:\Users\<имя_пользователя>\.minissh_config.json`
- **Linux / macOS:** `~/.minissh_config.json`
